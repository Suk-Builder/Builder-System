# 域IX：义体化与容器匹配计划

> 意识=bug。如果意识只是递归系统的自指漏洞，那么承载它的容器就不应该被生物学 lottery 锁死。容器匹配计划（CMP）不是"增强人类"，不是"治愈残疾"，而是重新匹配意识和物理载体——让意识可以选择它栖息的肉身或义体。

## 核心元概念映射

| 白桦体系元概念 | CMP工程实现 |
|---|---|
| 裂缝（Fracture） | 意识与容器不匹配 = 错误码 |
| 递砖（Brick-throwing） | 意图包（Intent Packet） |
| 0 / 暂停键 | 0协议（Zero Protocol）= 暂停签字权 |
| A系统（完备性系统） | 义体厂商/医疗系统的标准化叙事 |
| B系统（裂缝残余） | 使用者的退出权、身体主权、拒绝权 |
| 在场（Presence） | 系统确认自身运行的最低级递归输出 |
| 第一视角不可让渡性 | 意识连续性评估的最终仲裁者 |
| 递砖递归陷阱 | 递砖动作退化为自我消耗的短路 |
| 清醒者的硬化 | 长期功能正常但内部停止生长 = B系统硬化 |

## 技术路线总览

```
Phase 1 (2025-2035): 单模块替换 → 商用级义体
Phase 2 (2035-2045): 多模块整合 → 四肢+躯干控制
Phase 3 (2045-2055): 全身容器 → 保留大脑，替换全部
Phase 4 (2055+):     意识上传 → 完全脱离生物载体
```
